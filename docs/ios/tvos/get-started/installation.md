# <a name="installing-tvos-support"></a>Установка поддержки tvOS

Apple выпустила Apple TV 4K и tvOS 11. Платформа Apple TV доступен для разработчиков, позволяя им для создания многофункциональных, насыщенные приложения и освободить их в магазине Apple большом встроенных приложений.

## <a name="xamarintvos"></a>Xamarin.tvOS

Чтобы начать работу с помощью Xamarin для разработки приложений для tvOS, установите последнюю версию Xamarin.iOS в Visual Studio для Mac или Visual Studio 2015 или 2017 г. в Windows.  

Xamarin.iOS 11 и более поздние версии поддерживают tvOS 11 SDK, который поставляется с 9 Xcode компании Apple. 

- [Заметки о выпуске для Xamarin.iOS](https://developer.xamarin.com/releases/ios/)
- [Заметки о выпуске для Xcode](https://developer.apple.com/library/content/releasenotes/DeveloperTools/RN-Xcode/Chapters/Introduction.html#//apple_ref/doc/uid/TP40001051-CH1-SW876)

## <a name="installing-tvos-support"></a>Установка поддержки tvOS

Чтобы установить Xamarin.tvOS:

1. **Установите последние Xcode** — [загрузить последнюю версию Xcode](https://developer.apple.com/xcode/download/) и установить его. Без Xcode установлен его будет невозможно для построения приложения Xamarin. 
2. **Запустите Xcode** — после установки Xcode, запустить его один раз до обновления и запуска Visual Studio для Mac. Xcode установит некоторые средства, необходимые для Xamarin.
3. **Установите последнюю версию Xamarin стабильный** -обновление до последней версии [стабильной версии Xamarin](https://developer.xamarin.com/recipes/cross-platform/ide/change_updates_channel/).

Теперь все готово для создания приложений Xamarin.tvOS.
