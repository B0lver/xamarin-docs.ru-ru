# <a name="installing-tvos-support-in-xamarin"></a>Установка поддержки tvOS в Xamarin

> [!TIP]
> Чтобы начать эксперименты с поддержкой предварительной версии Xamarin для iOS 12 и tvOS 12, ознакомьтесь с [руководством по началу работы с iOS 12](~/ios/platform/introduction-to-ios12/get-started.md).

Компания Apple выпустила Apple TV 4 и tvOS 11. Платформа Apple TV открыта для разработчиков, позволяя создавать полнофункциональные впечатляющие приложения и освобождать их с помощью встроенного магазина приложений Apple TV.

Xamarin. iOS 11 и более поздней версии поддерживают пакет SDK для tvOS 11, поставляемый с компанией Apple Xcode 9.

- [Заметки о выпуске для Xamarin. iOS](https://docs.microsoft.com/xamarin/ios/release-notes/)
- [Заметки о выпуске для Xcode](https://developer.apple.com/library/content/releasenotes/DeveloperTools/RN-Xcode/Chapters/Introduction.html#//apple_ref/doc/uid/TP40001051-CH1-SW876)

## <a name="installation"></a>Установка

Для создания приложений tvOS с помощью Xamarin:

1. **Установка последней версии Xcode** — [скачайте последнюю версию Xcode](https://developer.apple.com/xcode/download/) и установите ее. Без установки Xcode невозможно будет создавать приложения Xamarin. 
2. **Запустите Xcode** — после установки Xcode запустите его один раз перед обновлением и запуском Visual Studio для Mac. Xcode установит некоторые средства, необходимые Xamarin.
3. **Установите последнюю стабильную версию Xamarin** — обновление до последней [стабильной версии Xamarin](https://github.com/xamarin/recipes/tree/master/Recipes/cross-platform/ide/change_updates_channel).
