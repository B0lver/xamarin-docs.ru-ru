# <a name="searching-data-in-the-cloud"></a>Поиск данных в облаке

_Поиск Azure — облачная служба, которая предоставляет возможности для отправляемых данных запросов и индексирования. Это устраняет требования к инфраструктуре сложности алгоритма поиска, традиционно ассоциирующиеся с реализацией функции поиска в приложении._

## <a name="searching-data-with-azure-searchazure-searchmd"></a>[Поиск данных с помощью поиска Azure](azure-search.md)

В этой статье показано, как использовать библиотеку поиска Microsoft Azure для интеграции службы поиска Azure в приложения Xamarin.Forms.
