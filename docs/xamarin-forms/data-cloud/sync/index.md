# <a name="synchronizing-data-with-web-services"></a>Синхронизация данных с веб-службами

_Автономная синхронизация позволяет пользователям взаимодействовать с мобильного приложения, просмотр, добавление или изменение данных, даже в том случае, где нет сетевого подключения. Изменения сохраняются в локальной базе данных, и когда устройство подключено к сети, можно синхронизировать изменения с веб-службой._

В этой статье описывается добавление автономной синхронизации функциональные возможности в Xamarin.Forms приложения.
