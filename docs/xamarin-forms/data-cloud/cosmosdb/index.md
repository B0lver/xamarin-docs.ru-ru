# <a name="storing-data-in-a-document-database"></a>Хранение данных в базе данных документа

_База данных документа Azure Cosmos DB является NoSQL базы данных, которая предоставляет высокоскоростной доступ к документам JSON, предоставляющие доступ к службе быстрое высокодоступные, масштабируемые базы данных для приложений, требующих удобный масштабирования и глобальные репликации._

## <a name="consuming-an-azure-cosmos-db-document-databaseconsumingmd"></a>[Использование базы данных служб Azure Cosmos DB документа](consuming.md)

В этой статье описывается использование клиентской библиотеки Microsoft Azure DocumentDB интегрировать базе данных Azure Cosmos DB документа в приложении Xamarin.Forms.

## <a name="authenticating-users-with-an-azure-cosmos-db-document-databaseauthenticationmd"></a>[Проверка подлинности пользователей с базой данных Azure Cosmos DB документа](authentication.md)

В этой статье объясняется, как для объединения с секционированных коллекций контроля доступа, чтобы пользователь имеет доступ только к своих собственных документов в приложении Xamarin.Forms.
