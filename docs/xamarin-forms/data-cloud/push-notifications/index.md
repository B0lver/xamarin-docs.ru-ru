# <a name="sending-push-notifications"></a>Отправка push-уведомлений

_Push-уведомления используются для доставки сведения, такие как сообщения, из серверной системы, приложения на мобильном устройстве, чтобы увеличить приложения вовлеченности и использования. Уведомления могут отправляться в любое время, даже в том случае, если пользователь не использует активно целевому приложению._

## <a name="sending-push-notifications-from-azure-mobile-appsazuremd"></a>[Отправка Push-уведомлений из мобильных приложений Azure](azure.md)

Концентраторы уведомлений Azure предоставляют масштабируемые Push-инфраструктуру для отправки мобильные Push-уведомления из любой серверной части на любую мобильную платформу, устраняя сложности серверной части, необходимость связываться с разных систем уведомлений платформы.
