# <a name="ios-platform-features"></a>Возможности на платформе iOS

## <a name="ios-specific-formatting"></a>Форматирование конкретных операций ввода-вывода

Xamarin.Forms позволяет межплатформенный пользовательский интерфейс стили и цвета, чтобы задать -, но существуют другие параметры для настройки темы iOS с помощью API для конкретных платформ в проекте iOS.

[Дополнительные сведения](theme.md) о форматировании пользовательский интерфейс, с помощью API-интерфейсам конкретной операций ввода-вывода, например **Info.plist** конфигурации и `UIAppearance` API.
