# <a name="no-locxamarinforms-pages"></a>:::no-loc(Xamarin.Forms)::: Pages

[![Загрузить образец](~/media/shared/download.png) загрузить пример](https://docs.microsoft.com/samples/xamarin/xamarin-forms-samples/formsgallery/)

_:::no-loc(Xamarin.Forms):::Страницы представляют собой кросс-платформенные экраны мобильных приложений._

Все типы страниц, описанные ниже, являются производными от :::no-loc(Xamarin.Forms)::: [`Page`](xref::::no-loc(Xamarin.Forms):::.Page) класса. Эти визуальные элементы занимают весь экран или большую часть экрана. `Page`Объект представляет в `ViewController` iOS и `Page` в универсальная платформа Windows. На Android каждая страница занимает экран `Activity` , как, но :::no-loc(Xamarin.Forms)::: страницы *не* являются `Activity` объектами.

[![::: No-Loc (Xamarin. Forms)::: типы страниц](pages-images/pages-sml.png)](pages-images/pages.png#lightbox "::: No-Loc (Xamarin. Forms)::: типы страниц")
