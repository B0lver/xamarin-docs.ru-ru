Заметки о тестировании
Xamarin.iOS
  • Самый простой способ начать работу — воспользоваться симуляторами iPhone, iPad, Apple Watch и Apple TV, включенные в Xcode. Доступ к симуляторам можно получить на подключенном компьютере Mac при отладке с помощью Visual Studio.
  • Чтобы использовать устройство для тестирования, следуйте этим инструкциям (большинство шагов выполняется на подключенном компьютере Mac).
Xamarin.Android Чтобы настроить устройство или использовать симулятор, следуйте этим инструкциям:
  • Установщик Xamarin включает диспетчер эмуляторов Google, который позволяет настроить эмуляторы Google Android для тестирования.
  • Другие эмуляторы предоставляются Genymotion и Майкрософт (вместе с Visual Studio 2015).
Xamarin.Forms Приложения Xamarin.Forms можно развертывать на соответствующих устройствах и платформах, как описано выше. Приложения iOS можно тестировать только на подключенном оборудовании Mac. Приложения Windows (классические и для планшетов) для Windows 8.1 или UWP можно тестировать непосредственно на компьютере разработчика.
