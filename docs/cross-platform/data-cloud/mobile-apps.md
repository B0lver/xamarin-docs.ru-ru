# <a name="xamarin-mobile-apps-with-azure-and-app-center"></a>Мобильные приложения Xamarin в Azure и центре приложений

Разработчики Xamarin могут воспользоваться преимуществами широкого спектра облачных служб, от сборки непрерывной интеграции с помощью App Center и машинного обучения с помощью Azure. [Скачайте этот плакат](https://docs.microsoft.com/xamarin/guides/cross-platform/azure/Resources/poster.pdf) или перейдите по ссылкам ниже, чтобы получить дополнительные сведения.

## <a name="app-center"></a>Центр приложений

Центр приложений Visual Studio поддерживает комплексные и интегрированные службы для разработки мобильных приложений. Разработчики могут использовать службы **сборки**, **тестирования** и **распространения** для настройки конвейера непрерывной интеграции и доставки. После развертывания приложения разработчики могут отслеживать состояние и использование своих приложений с помощью служб **аналитики** и **диагностики** , а также привлекать пользователей с помощью службы **push-уведомлений** . Разработчики также **могут использовать проверку подлинности** для аутентификации пользователей и службы **данных** , чтобы сохранять и синхронизировать данные приложений в облаке.

Если вы хотите интегрировать облачные службы в приложения Xamarin, [посетите документацию](https://docs.microsoft.com/appcenter) и зарегистрируйтесь в [центре приложений](https://appcenter.ms/signup?utm_source=XamarinDocs&utm_medium=Azure&utm_campaign=docs) уже сегодня.

## <a name="azure"></a>Azure

Сведения о [разработке мобильных приложений с помощью облачных служб](https://docs.microsoft.com/azure/mobile-apps/), включая SignalR, программы для обучения, машинное обучение, пространственные привязки, Поиск и многое другое.
