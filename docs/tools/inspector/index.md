# <a name="xamarin-inspector"></a>Xamarin Inspector

Инспектор Xamarin интегрируется с отладки интегрированной среды разработки, что позволяет интерактивно выполнить код C# или проверить свойства пользовательского интерфейса, обслуживающий отладки или диагностики помощи при проверке запущенного приложения рабочий процесс приложения.

Проверка работающем приложении доступен для корпоративных клиентов. Сообщите нам, нужна ли вам [возникли ошибки](~/tools/inspector/install.md#reporting-bugs). У нас есть [форум](https://forums.xamarin.com/categories/inspector) слишком!
