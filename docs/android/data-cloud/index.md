# <a name="data-and-cloud-services"></a>Данные и облачные службы

## <a name="data-and-cloud-services"></a>Данные и облачные службы

Приложениям Xamarin. Android часто требуется доступ к данным (из локальной базы данных или из облака), а многие из этих приложений используют веб-службы, реализованные с использованием широкого спектра технологий. В руководстве в этом разделе рассматриваются способы доступа к данным и использования облачных служб.

## <a name="data-accessandroiddata-clouddata-accessindexmd"></a>[Доступ к данным](~/android/data-cloud/data-access/index.md)

В этом разделе обсуждается доступ к данным в Xamarin. Android с помощью SQLite в качестве ядра СУБД.

## <a name="google-messagingandroiddata-cloudgoogle-messagingindexmd"></a>[Обмен сообщениями Google](~/android/data-cloud/google-messaging/index.md)

Google предоставляет как Firebase облачные системы обмена сообщениями, так и устаревшие службы Google Cloud Messaging для облегчения обмена сообщениями между мобильными приложениями и серверами приложений. В этом разделе содержатся обзоры для каждой службы, предоставляемые пошаговым описанием использования этих служб для реализации удаленных уведомлений (также называемых Push-уведомлениями) в приложениях Xamarin. Android.
